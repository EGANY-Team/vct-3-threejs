Demo: https://vct-3-threejs.now.sh

## Cài đặt

```bash
# clone repo
git clone https://github.com/EGANY-Team/vct-3-threejs

# install
cd vct-3-threejs
npm install

# start server at http://localhost:1234/
npm start

# build
npm run build
```
